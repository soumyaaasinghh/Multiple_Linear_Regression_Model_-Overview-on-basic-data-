There is a dataset provided named as 50_Startups.csv And an .ipynb file which basically step by step creates Multiple Regression Model based on the given dataset. Starting from importing the required libraries then importing the dataset after that encoding the categorical data (here the categorical data is the STATE column) & dividing that dataset into Training set (X_train & y_train) & Test set (X_test, y_test) then Trained that Model on the training set and also predicted the results of the test set.
In the last step where we have to predict, we havent used the 'plot' function unlike in the Simple Linear Regression Model because it has more than one independent variable and we cannot show it on the chart having 2D dimension therefore we require 2 vectors - one is for for the predicted profit (y_pred) of the test set and another one is for the actual profit (y_test) of the test set
